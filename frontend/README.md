## Frontend Migration (React)

### 1. Verify Node.js is installed
node -v
npm -v

### 2. Start FastAPI backend
From project root:
uvicorn api:app --reload
Backend runs at http://localhost:8000 so keep this terminal open.

### 3. Install Frontend Dependencies
cd frontend
npm install

### 4. Run dev server
npm run dev

### 5. Open in browser
http://localhost:5173
