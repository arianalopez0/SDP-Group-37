## Frontend Migration (React)

### 1. Verify Node.js is installed
```bash
node -v
npm -v
```md
### 2. Install Frontend Dependencies
```bash
cd frontend
npm install
```md
### 3. Run dev server
```bash
npm run dev
````md
### 4. Open in browser
http://localhost:5173


The React frontend currently runs in mock mode using real Streamlit schema output.
